# Туристическое приложение.
<b>Цель разработки:</b>
<br>Создания приложения, в котором пользователи смогут разнообразить свой отдых за счет выбора широкого количества услуг развлекательного характера, сведенных в одно удобное приложение. Благодаря этому возможно развитие туризма в различных регионах.

<b>Инструменты, применяемые при разработке</b>
<br>Целевая платформа: 
<br>Android версии 8.0 (Oreo) и выше
<br>Библиотека для построения приложения: Kotlin
<br>Хранение данных: локально.

<br><b>Описание приложения</b>
1.	<b>Экран профиля</b>
<br>Профиль приложения должен содержать как основную информацию о пользователе, возможность редактировать данную информацию (имя/фамилию, фотографию), так и следующие функции:
-просмотр сохраненных объявлений. Пользователь должен иметь возможность сохранить понравившуюся ему активность.
-Выход. Выход из под данной учетной записи приложения.
2.	<b>Экран «карта»</b>
<br>На данном экране размещены предложения об услугах, расположенные на карте. Таким образом, пользователь может выбрать интересующую его услугу при этом, расположенную в удобном ему месте. При нажатии на конкретную кнопку, открывается пре-просмотр карточки услуги, на которой указан адрес, а также, краткое описание услуги. Из данного пре-просмотра можно перейти как в полную карточку услуги, так и вернуться назад. 
3.	<b>Карточка услуги</b>
<br>В карточке услуги представлена максимально подробная информация о предоставляемой услуге: точное местоположение, стоимость, описание, а также, возможность перейти на сайт, позвонить организатору услуги, а также, проложить маршрут до места оказания данной услуги. 
Также, в ТОПе дается возможность сохранить данное объявление.
